Install WaPulse MCP Server in Cursor

Option 1: NPX Installation (Recommended)

Use this configuration for direct NPX installation:

{
  "wapulse-whatsapp": {
    "command": "npx",
    "args": [
      "-y",
      "wapulse-whatsapp-mcp-server@latest"
    ],
    "env": {
      "WAPULSE_TOKEN": "your-wapulse-token-here",
      "WAPULSE_INSTANCE_ID": "your-instance-id-here"
    }
  }
}

Before clicking the install link:

  1. Replace your-wapulse-token-here with your actual WaPulse API token
  2. Replace your-instance-id-here with your actual WaPulse instance ID

Option 2: Smithery Cloud

For Smithery cloud deployment (requires Smithery account):

{
  "wapulse-whatsapp-smithery": {
    "command": "npx",
    "args": [
      "-y",
      "@smithery/cli@latest",
      "run",
      "@Quegenx/wapulse-whatsapp-mcp",
      "--key",
      "your-smithery-key",
      "--profile",
      "your-profile"
    ]
  }
}

Manual Installation

Alternatively, you can manually add this to your ~/.cursor/mcp.json file:

{
  "mcpServers": {
    "wapulse-whatsapp": {
      "command": "npx",
      "args": [
        "-y",
        "wapulse-whatsapp-mcp-server@latest"
      ],
      "env": {
        "WAPULSE_TOKEN": "your-wapulse-token-here",
        "WAPULSE_INSTANCE_ID": "your-instance-id-here"
      }
    }
  }
}

Getting Your Credentials

  1. Visit WaPulse.com
  2. Create an account and get your API token
  3. Create a WhatsApp instance and note the instance ID
  4. Replace the placeholder values in the configuration above
